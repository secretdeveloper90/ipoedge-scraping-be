IPO Edge API

This is the backend API for IPO Edge Panel application.

API endpoints are available at:

  • /api/health - Health check
  • /api/ipos/health - IPO service health check
  • /api/ipos/listing-details - Get listing details
  • /api/ipos/company/:companyId - Get company details
  • /api/ipos/screener/:year - Get screener data
